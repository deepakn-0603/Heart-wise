# **App Name**: HeartWise

## Core Features:

- Patient Data Input: Collect patient details through a user-friendly form including age, sex, cholesterol, blood pressure, and other relevant medical data.
- Heart Disease Risk Prediction: Utilize a pre-trained machine learning model (e.g., Logistic Regression, Random Forest) as a tool to predict the risk of heart disease based on the input data.
- Risk Assessment Display: Display the prediction result, indicating whether the patient is at risk (yes/no) and the associated probability.
- Data Visualization: Provide a visual representation of health data using charts to help users understand their risk factors better.
- Secure Authentication: Implement secure user registration and login functionality to protect patient data.
- Diagnosis History: Enable users to view their previous diagnosis results for comparison and monitoring purposes.

## Style Guidelines:

- Primary color: Calm blue (#64B5F6) to inspire trust and reliability.
- Background color: Light gray (#F0F4F8) for a clean, clinical feel.
- Accent color: Soft green (#A5D6A7) for positive confirmations and highlights.
- Body and headline font: 'PT Sans', a modern and readable sans-serif font for headlines and body text.
- Use clean, simple icons related to health metrics and data visualization.
- Maintain a clean and organized layout with clear visual hierarchy to make the application easy to navigate.
- Use subtle transitions and animations to enhance user experience and provide feedback on interactions.
